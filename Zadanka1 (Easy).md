**Exercise: The Tip Calculator (Variables & Math)**

**Goal: Practice basic arithmetic and f-string formatting**

- Create a variable bill_amount and set it to 50.0.

- Create a variable tip_percentage and set it to 0.15 (15%).

- Calculate the total bill (bill + tip) and store it in a variable called total.

- Print the result as: "The total bill is $57.5".

**## Exercise: The Secret Password (While Loops)**

**### Goal: Introduce the concept of a loop that runs until a condition is met.**

- Create a variable secret and set it to "python123".
- Use a while loop to keep asking the user to "Enter the password:".
- If they guess correctly, print "Access Granted" and break the loop.
- If they guess wrong, print "Try again!".

**## Exercise: The Mad Libs Generator (Strings & Input)**

**### Goal: Practice string concatenation and f-strings.**

- Use an f-string to combine these into a funny sentence.

**## Exercise: The Vowel Counter (Strings & Loops)**

**### Goal: Iterate through a string and check for specific characters.**

- Create a variable text with a sentence of your choice.
- Create a variable count and set it to 0.
- Use a for loop to look at every character in the string.
- If the character is a vowel (a, e, i, o, u), increment the count.
- Print the total number of vowels found.

**## Exercise: The Student Ranker (Lists & Enumerate)**

**### Goal: Use the position of an item in a list to provide context.**

- Given a list of names: ["Alice", "Bob", "Charlie", "Diana"].
- Use a for loop with `enumerate` to print their rank.
- The output should look like:
    Rank 1: Alice
    Rank 2: Bob
    ...

    Hint: Remember that enumerate starts at index 0 by default. How do you make it start at 1?
